# `ViewModelsProvider` Component   


### Usage  

```tsx
import { ViewModelStoreBase, ViewModelsProvider } from "mobx-view-model";

const vmStore = new ViewModelStoreBase();

export const App = () => {
  return (
    <ViewModelsProvider value={vmStore}>
      ...
    </ViewModelsProvider>
  )
}
```
